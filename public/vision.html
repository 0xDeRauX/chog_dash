Communautés — Discord · mentions · prix

Uniquement les actifs qui ont les trois données. Membres Discord, mentions X et prix, indexés base 100 pour comparer les dynamiques de communauté.

Coefficients de corrélation

Corrélation de Pearson sur la fenêtre choisie (les valeurs, comme le nuage de points). +1 = varient ensemble, −1 = à l'opposé, 0 = pas de lien. Un trend commun peut la gonfler — le nuage sert de contrôle visuel. Les paires avec les mentions restent vides tant que l'historique est court.

Nuage de points — Prix vs TVL

Chaque point = un jour (prix et TVL indexés base 100). Un nuage aligné en diagonale = forte corrélation.

Leaderboard

Clique sur une colonne pour trier (plus haut / plus bas). Scopé sur l'univers sélectionné plus haut.
