Collecte manuelle — relance & backfill

Le site est statique : les collecteurs tournent dans GitHub Actions. Cette section déclenche le workflow collect-manual à distance — pour relancer une collecte ratée la veille, ou backfiller les mentions X sur N jours. À la fin du run, data.json est commité et déployé automatiquement (~2 min), il suffit de recharger la page.
